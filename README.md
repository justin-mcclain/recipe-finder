# Recipe Finder

This app allows you to enter some ingredients you have in your pantry and find a recipe to make with them.

## Technology Used

This project utilized React, Material UI

### `npm run build`

Builds the app for production to the `build` folder.\

The build is minified and the filenames include the hashes.\
